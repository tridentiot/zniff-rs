# Learning Rust
The zniff.rs project is also about learning Rust.

# Pros and Cons
This is a list of pros/cons discovered during development.

## Vs. C

### Pros
- Enums can contain fields like structs. Great for returning data with an enum.
- All enums MUST be handled in a match (switch-case in C).
- Variables are constant (immutable) by default.
- All fields in a struct or enum MUST be initialized.
- A function pointer is a dedicated type.
- Error propogation using '?'.

### Cons
- Steep learning curve
