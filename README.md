# zniff.rs

This is a from-the-scratch Rust implementation of a Z-Wave packet sniffer service.

Use cases:
- https://github.com/tridentiot/z-wave-test-system
- https://github.com/tridentiot/z-wave-pc-zniffer

# Usage

```bash
zniff-rs --help
```

# Development

```bash
cargo build
```

```bash
cargo run -- --help
```
